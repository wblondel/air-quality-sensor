This project was successfully built with :
- [ESP-IDF](https://github.com/espressif/esp-idf) [v5.4.4](https://github.com/espressif/esp-idf/tree/v5.4.4)
- [ESP-Matter](https://github.com/espressif/esp-matter) [@823435b](https://github.com/espressif/esp-matter/tree/823435bd69eaa49f4af8e82fe8ce79ad743b164a)

on a MacBook Pro M1 running macOS Golden Gate 27.0 with :
- cmake 4.3.4
- ninja 1.12.1
- Python 3.14
- Git 2.54
- dfu-util 0.11
- ccache 4.13.6
