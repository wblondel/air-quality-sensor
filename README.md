- [ESP-Matter](https://github.com/espressif/esp-matter) [@080c9e7](https://github.com/espressif/esp-matter/tree/080c9e7efc00edf6431a286870ae6cf24fa1f96f)

on a MacBook Pro M1 running macOS Sequoia 15.5 with :
- cmake 4.0.2
- ninja 1.12.1
- Python 3.13.5
- Git 2.49.0
- dfu-util 0.11
- ccache 4.11.3_1
